💡 臨床速記：成分相同，強度不同？

Ointment > Cream > Lotion

在成分與濃度完全相同的情況下，軟膏 (Ointment) 因封閉性強、穿透力佳，其強度分級通常比乳膏 (Cream) 高出一級。例如：Betamethasone dipropionate 軟膏是 Class 2，但乳膏是 Class 3。

Ref: Stoughton-Cornell Classification System; National Psoriasis Foundation.

🎯 臨床策略 (Strategy)

  • 乾燥苔蘚化 (Lichenified)

    首選 軟膏。可配合保鮮膜封閉療法 (Occlusion) 增加 10-100 倍吸收。

  • 急性滲出 (Weeping)

    首選 乳膏 或濕敷。避免油膏導致阻塞毛囊。

  • 頭皮 (Scalp)

    首選 泡沫 或 洗劑。病人依從性最高。

📉 停藥策略 (Tapering)

長期使用中強效類固醇後，驟然停藥可能導致反彈 (Rebound)。應採取漸進式停藥。
策略 A：降低頻率 (Frequency)

最常用的方法。

每日2次 → 每日1次 → 隔日1次 → 週末療法
策略 B：降低強度 (Potency)

適合大面積或兒童。

Class 2 (軟膏) → Class 4 (乳膏) → Class 7
策略 C：週末療法 (Weekend)

對於慢性反覆發作者，在病灶緩解後，每週僅在週六、週日塗抹一次藥膏，平日只擦乳液。可顯著降低復發率且副作用極低。

💧 濕敷療法 (Wet Wrap)

適用於嚴重、頑固型異位性皮膚炎。可增加藥物吸收、保濕並冷卻止癢。
1
浸泡與塗藥 (Soak & Seal)

洗澡後輕拍乾(微濕)，立即厚塗類固醇藥膏於病灶，其餘部位塗保濕劑。

2
濕層包覆 (Wet Layer)

將管狀繃帶或棉質衣物用溫水浸濕後擰乾，穿/套在身上。

3
乾層固定 (Dry Layer)

外層再穿一層乾燥的衣物或繃帶固定。

🤔 臨床迷思：為何不用強效藥？

1. 吸收倍增： 濕敷的「封閉效應」會使藥物穿透率暴增 10~100倍。使用強效藥極易造成全身性副作用。

2. 安全平衡： 使用 Class 4-7 (中弱效) 藥膏搭配濕敷，能達到「局部強效治療」的效果，但保持吸收風險在安全範圍內。通常限制 3-5 天。

Ref: AAD Eczema Action Plan; Wet wrap therapy in children with moderate to severe atopic dermatitis (AAAAI).

  • 避免使用 Class 1-3 強效藥，除非醫師特別指示。

🛡️ 副作用監測與處置
皮膚萎縮 (Atrophy)

長期使用強效藥最常見的局部副作用。通常停藥後可逆，但產生皺紋可能無法完全恢復。

類固醇酒糟/痤瘡

臉部長期不當使用中強效類固醇所致。特徵是停藥後會惡化 (Rebound)，需耐心治療。

全身性吸收

長期大面積使用 Class 1 藥膏可能抑制 HPA 軸，導致庫欣氏症候群 (Cushing's)，需定期監測。

🧪 複方藥膏 (Combination Therapy)

1. 水楊酸 (Salicylic Acid)
  • 機制：角質溶解 (Keratolytic)，軟化並去除增厚的角質層，幫助類固醇穿透。
  • 適應症：厚皮型乾癬、嚴重苔蘚化濕疹、掌蹠角化症。
  • 注意：大面積使用可能造成水楊酸中毒 (Salicylism)，兒童應慎用。
2. 尿素 (Urea)
  • 機制：增加角質層含水量 (Hydration) 並具有輕微去角質效果。
  • 適應症：極度乾燥的濕疹、魚鱗癬、毛孔角化症。
  • 優勢：比水楊酸溫和，保濕效果更佳。
3. 抗生素/抗黴菌劑 (Antimicrobials)
  • 成分：常見如 Neomycin, Gentamicin, Clotrimazole。
  • 適應症：濕疹合併金黃色葡萄球菌感染 (流湯/結黃痂)，或黴菌感染急性發炎期。
  • 警示：一旦發炎緩解應停用複方，改回單方抗感染藥，以免產生抗藥性或導致難治癒癬病 (Tinea incognito)。

👶 特殊族群 (Special Pops)

🧸 嬰幼兒 (Children)

皮膚薄，體表面積比例大，容易全身性吸收。首選 Class 6-7 (Mild)。避免強效藥膏，避免大面積使用。

🤰 懷孕 (Pregnancy)

FDA 分級通常為 Category C。弱效與中效 (Mild/Moderate) 被認為相對安全。應避免大量使用強效藥膏，因可能與低出生體重有關。

👁️ 臉部/眼周 (Face/Eyes)

眼周長期使用可能導致青光眼或白內障。務必限制在 Class 7 且極短期使用。

Ref: WHO Model Prescribing Information: Drugs Used in Skin Diseases.
